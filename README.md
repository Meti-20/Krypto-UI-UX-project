# KRYPTO – NFT Marketplace Landing Page UI/UX

A modern NFT marketplace landing page designed in **Figma**, focusing on a clean interface, intuitive user experience, and responsive layout.

## 🚀 Overview

This project explores modern UI/UX principles for a cryptocurrency/NFT platform, emphasizing usability, visual hierarchy, and a consistent design system.

## 🛠️ Tools

* Figma
* Auto Layout
* Components & Variants
* Prototyping

## ✨ Highlights

* Modern landing page
* Responsive layout
* Clear user flow
* Consistent design system
* High-fidelity prototype
